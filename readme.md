## 개요 - https://www.hanggok.com
- 서버 접속시 보게되는 화면
- 현재 초기 버전으로, 추후 여러 기능을 추가할 예정.

## 구동 환경
- **Raspberry Pi 3B**의 하드웨어 및 **Raspberry Pi OS**를 사용함.
- Docker를 통하여 **Nginx Proxy Manager (NPM)** 을 구동함.
- 웹서버는 **리버스 프록시**를 통해 구현되고, **Let's Encrypt 인증서**를 사용하여 **HTTPS 보안**이 적용됨.

## 추후 확장 계획
- 현재 저성능의 arm기반 서버에서 구동하여 다양한 서비스 제공에 어려움이 있음.
- 향후 **x64 기반 서버**로 시스템을 이동하여 **파일 서버, 웹서버, 일부 게임 서버**를 지원할 계획이 있음.
